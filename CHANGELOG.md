# 변경 기록

## 0.1.1 - 2026-07-11

### 개선

- Obsidian 미리보기 DOM의 계산된 스타일, 가상 요소, 표, 콜아웃, 인라인 배경과 장식을 브라우저가 직접 그리는 페이지 스냅샷 경로를 추가했습니다.
- `::before`, `::after`, `::marker`, `::first-letter`, `::first-line`과 선행 구두점, 생성 텍스트, 보이는 `aria-hidden` 콘텐츠를 보존합니다.
- 병합 셀과 `col`/`colgroup` 너비, 변형·중첩 스크롤 영역의 sticky/absolute/fixed 요소, 수직 쓰기·ruby·다단·클리핑을 더 정확히 고정합니다.
- 한글 자모, 옛한글, 결합 문자, ZWJ, 국기 문자를 grapheme 단위로 측정하고 실제 DOM 좌표 폭 안에만 선택 텍스트를 배치해 글자 겹침을 줄였습니다.
- 인라인 태그 사이의 실제 공백과 `normal`, `nowrap`, `pre`, `pre-wrap`, `pre-line`, CSS4 공백 모드의 탭·연속·앞뒤 공백을 선택 텍스트에 보존합니다.
- 비균일 페이지 구간은 한 번만 캡처하고 남는 페이지 영역을 배경으로 채워 페이지 경계 중복을 방지합니다.
- 로컬 이미지, Canvas, SVG, CSS와 글꼴 리소스에 항목별·누적·시간·크기 상한을 적용하고, 안전하게 스냅샷할 수 없으면 호환 렌더러로 전환합니다.
- 긴 문서는 모든 페이지가 PDF 안에서 동시에 보유하는 래스터 메모리까지 계산해 공통 해상도를 안전하게 낮추며, 최소 판독 배율을 지킬 수 없으면 할당 전에 중단합니다.
- 인라인 서식, 복합 표, 콜아웃, 코드, 미디어, 다크 테마, 흑백, 2배 해상도, 비균일 페이지와 장문 문서를 다루는 Chrome·WKWebView·PDFKit 회귀 fixture를 추가했습니다.

### 유지되는 제한

- 선택형 PDF는 화면 이미지와 보이지 않는 선택 텍스트 레이어를 결합하며 PDF/UA 문서가 아닙니다.
- 다른 플러그인이 늦게 갱신하는 내용은 캡처 시점에 따라 달라질 수 있습니다.
- WebView가 DOM 스냅샷을 안전하게 만들 수 없는 리소스나 매우 큰 문서는 제한된 호환 렌더러를 사용합니다.
